# Sortoi GUI

A modern desktop interface for Sortoi, built with Electron, React, and Vite.

## Features

- **Visual Control**: Manage your file organization with a clean, modern interface.
- **Real-time Progress**: Watch as the AI analyzes and organizes your files.
- **Detailed Preview**: Review every proposed file move before it happens.
- **Session History**: Easily browse past organization sessions and rollback if needed.

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```
